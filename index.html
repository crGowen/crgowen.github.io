Stenor

Short for "Steganographer", Stenor is a hobby project on digital steganography, imitating the cyber-spy thriller cliché of secretly storing data inside of data. The tool takes an input file, and as a second input an image or audio file, and encodes the input file into the image or audio without changing it in any noticable way. The hidden data can then be decoded and extracted by Stenor.

Stenor is available to download here.

Fast Latin-to-Russian Characters

Firefox extension which, when enabled, will automatically convert Latin phonetics to Russian alphabet characters as you type. Example: 'pozhaluysta' converts to 'пожалуйста'.
